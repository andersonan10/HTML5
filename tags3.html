*HEADER Tag Semântica para cabeçalho da página; Podemos te mais de uma tag HEADER na página mas com contextos diferentes; 1 HEADER para cabeçalho da página e outro header para um artigo da mesma página; Exemplo:

Titulo do artigo

*MAIN Tag que define conteudo principal da página; So deve ter um Main na página; Exemplo:

Titulo do site

*FOOTER Define o rodapé da página; Pode ser usada em uma ARTICLE ou SECTION; Muito comum para links; Exemplo:
  • Home
  • Categoria
  • Contato

*SECTION Usada para definir uma sesão do site; Não usar a tag SECTION quando: - Se for uma área de NAVEGAÇÃO usar tag NAV; - Se for um CABEÇALHO usar tag HEADER; - Se for a área PRINCIPAL usar a tag MAIN; - Se for um ARTIGO usar a tag ARTICLE; - Se for um RODAPÉ usar a tag FOOTER; Caso não seja nenhuma dessas alternativas ai sim usar a tag SECTION ao invés de uma DIV; Exemplo:
