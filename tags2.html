*TABLE Utilizado para criar tabelas; Não recomendado para criar layouyt; Precisa utilizar as tags TR e TD para criar as linhas e colunas; Para usar uma tabela completa precisamos utilizar: caption, col, colgroup, thead, tfoot, tbody. Exemplo:
A1 B1 C1
A2 B2 C2
*TR (Table row) utilizado para criar linhas na tabela; Usar sempre de uma tag TABLE; *TD Usado para criar celulas nas linhas da tabela; Usar sempre dentro de uma tag TR; Atributos: colspan = Define numero de colunas que esta célula deve ocupar; headers = Define uma mais células de cabeçalho relacionadas; rowspan = Define o número de linhas que esta célula deve ocupar. Exemplo: //Aqui defino que A1 vai ocupar 2 celulas (A1 e A2) //Aqui defino que B2 vai ocupar 2 linhas (B2 e B3)
A1C1
A2 B2C2
A3 C3
