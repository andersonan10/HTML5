*SPAN Recomenda-se usar para aplicar CSS em algum texto; Não tem peso semântico; Caso precise mudar a aparência somente sem se preocupar com a semântica, o SPAN é recomendado. ; *LINK Utilizada para linkar para um arquivo externo como CSS, HTML, também uma URL ou até mesmo um ícone; Usado dentro da tag HEAD; //Este link personaliza o ícone da barra do navegador; //Este link da uma versão alternativa do SITE, exemplo disso seria um site de galeria de fotos e a versão alternativa é em texto como JSON; //Este link acelera a solicitação para o site que eu apontar por DNS; //Quando sabemos qual página provavelmente o usuário irá depois da página HOME, podemos usar o NEXT para deixar carregado em background; //Funciona como o NEXT, carregando em background a página anterior; //Carrega o site externo, fazendo a conexão antes de carregar a própria página; //Tenta fazer o render de umapágina externa para deixar preparada quando minha página solicitar. *NOSCRIPT Serve para carregar um javascript alternativo para 2 situações: 1 - Quando o usuário desativar o javascript no navegador, ai será carregado o conteúdo do NOSCRIPT; 2 - Quando o navegador não tem suporte ao javascript do meu código padrão.
