Não tem peso semântico; Caso precise mudar a aparência somente sem se preocupar com a semântica, o SPAN é recomendado. ; *LINK Utilizada para linkar para um arquivo externo como CSS, HTML, também uma URL ou até mesmo um ícone; Usado dentro da tag HEAD; //Este link personaliza o ícone da barra do navegador; //Este link da uma versão alternativa do SITE, exemplo disso seria um site de galeria de fotos e a versão alternativa é em texto como JSON;
