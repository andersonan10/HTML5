*FORM Tag para criar formulários; Atributos: - ACCEPT-CHARSET = Define o encode usado ao enviar o formulário; - ACTION = Define para qual URL o formulário será enviado; - AUTCOMPLETE = Define se os campos terão autocompletar com ajuda do navegador; - ENCTYPE = Define qual codificação de formulário será usada; - METHOD = Define qual será o método de envio do formulário; - NAME = Define qual o nome do formulário; - NOVALIDATE = Define se deve ser validado ao enviar; - TARGET = Define qual o alvo ao enviar o formulário; Exemplos: //com o POST os dados não serão visualizados
*INPUT Tag usada para definir um campo de entrada; A tag pode variar de várias maneiras dependendo do atributo TYPE; Atributos: - ACCEPT = Define o tipo de arquivo aceito para ser selecionado; - ALT = Define um texto alternativo, aplicável a imagens; - AUTOCOMPLETE = Define se o campo pode ter habilitado o auto completar; - AUTOFOCUS = Define o foco neste campo quando a página é carregada; - CHECKED = Define se o campo está marcado/selecionado; - DIRNAME = Define o nome do campo que será enviado informando a direção do texto escrito; - DISABLED = Define o campo como desabilitado; - FORM = Define qual o formulário que esse campo pertence, mesmo o input estando dentro do FORM esse atributo enviará para outro formulário; - FORMACTION = Define qual a URL de envio desse campo, mesmo o input estando dentro do FORM esse atributo enviará para outro formulário; - FORMENCTYPE = Define qual o enctype do formulário ao enviar esse campo; - FORMMETHOD = Sobrescreve o method padrão do formulário; - FOMNOVALIDATE = Sobrescreve o atributo novalidate do form; - FORMTARGET = Define qual será o target do formulário ao clicar neste botão; - HEIGHT = Define a altura do campo; - LIST = Define qual tag datalist está vinculada com este campo; - MAX = Define um valor máximo, numeros e datas; - MAXLENGTH = Define um comportamento máximo de caracteres, numeros; - MIN = Define um valor mínimo, numeros e datas; - MULTIPLE = Define se o usuário pode entrar com mais de um valor neste campo; - NAME = Define o nome do campo; - PATTERN = Define uma expressão regular para validação do campo; - PLACEHOLDER = Define um texto de dica que descreve o valor esperado; - READONLY = Define o campo como somente leitura; - REQUIRED = Define o campo como obrigatório para envio do formulário; - SIZE = Define a largura, em caracteres do campo; - SRC = Define a fonte do arquivo; - STEP = Define o intervalo de número permitidos; - TYPE = Define o tipo do campo; - VALUE = Define o valor inicial do campo; - WIDTH = Define a largura do campo; Exemplos: // Só aceita arquivos .png // Aceita mais de 1 arquivo para fazer upload // Campo escondido // Campo de escolha de mês // Aceita somente números, e pula de 2 em 2 // Botão que limpa os dados do formulário // Caixa de busca do site // Horas // Campo que aceita links // Campo de calendário de semana
